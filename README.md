## Cara Menjalankan

```bash
# Install dependencies
pnpm install

# Jalankan server development
pnpm dev

# Build untuk produksi
pnpm build
```

## Deployment

Aplikasi ini siap dideploy ke Vercel, Netlify, atau Cloudflare Pages.
Untuk Cloudflare Pages, pastikan ada file `public/_redirects` untuk routing SPA.
